DISPLAY FLEX: Tenemos este texto pegado a la izquierda, y la botonera a la derecha. Centrar verticalmente. Primer cambio: - Modifica el comportamiento que hay dentro de la capa. Display flex se comporta igual que el display block. Display inline flex se comporta igual que display inline block. Con Display flex por defecto se convierte en horizontal. Si no lleva display flex no tiene sentido poner justify-content flex-wrap: son las capas que sobresalen de la propia capa. inline items contents: para horizontal align -items: para alineación vertical justify content es el que más utilizás. flex-direction: row (por defecto ya es así si no pones nada) opciones: row, column, row reverse, column reverse flex-flow junta el flex-direction y el flex-wrap, pero no se usa mucho. El width es el display flex en muchos casos no harán falta, pero en ciertos casos se aplicará.
